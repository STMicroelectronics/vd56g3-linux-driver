STMicroelectronics VD56G3 global shutter sensor

Required Properties:
- compatible: shall be "st,st-vd56g3"
- clocks: reference to the xclk input clock.
- clock-names: shall be "xclk".
- VCORE-supply: sensor digital core supply. Must be 1.1 volts.
- VDDIO-supply: sensor digital IO supply. Must be 1.8 volts.
- VANA-supply: sensor analog supply. Must be 2.8 volts.

Optional Properties:
- reset-gpios: reference to the GPIO connected to the reset pin, if any.
	       This is an active low signal to the vd56g3.

The device node must contain one 'port' child node for its digital output
video port, in accordance with the video interface bindings defined in
Documentation/devicetree/bindings/media/video-interfaces.txt.

vgxy61: csi2tx@10 {
	compatible = "st,st-vd56g3";
	reg = <0x10>;
	status = "okay";
	clocks = <&clk_ext_camera>;
	clock-names = "xclk";
	VCORE-supply = <&v1v1>;
	VDDIO-supply = <&v1v8>;
	VANA-supply = <&v2v8>;
	reset-gpios = <&mfxgpio 18 GPIO_ACTIVE_LOW>;
	port {
		ep0: endpoint {
			data-lanes = <1 2>;
			remote-endpoint = <&mipi_csi2_out>;
		};
	};
};
